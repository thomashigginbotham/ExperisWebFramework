Parameters

Parameter Default Value Available Values Description
els null array of dom elements

options.lines 4 number

The number of lines of text that will remain visible.
options.moreLabel 'More' string

The hyperlink text that will appear below the collapsed text
options.lessLabel 'Less' string

The hyperlink text that will appear below the expanded text
